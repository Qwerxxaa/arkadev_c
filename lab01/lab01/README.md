# arkadev_c  
Используемые технологии и библиотеки
 • Язык программирования: C
 • Компилятор: GCC
 • Библиотеки:
 • <stdio.h> — для ввода и вывода данных

Структура папки lab01

Файлы и описание заданий:
 • task01.c — Первая программа с переменными (int, float, char) и их вывод
 • task2.c — Вывод размеров типов (sizeof)
 • task3.c — Вывод значений переменных и их адресов в памяти
 • task4.c — Остаток от деления двух чисел с использованием %
 • task5.c — Инкремент и декремент (a++, ++a, a--, --a)
 • task6.c — Преобразование типов для точного деления
 • task7.c — Потеря точности при работе с float
 • task8.c — Переполнение типа int
 • task9.c — Вычисление формул: x^2 + 2x + 1 и (x + 1)^2 − (x − 1)^2
 • task10.c — Обмен значений двух переменных через третью переменную
 • task11.c — Вывод последней цифры целого числа
 • task12.c — Разложение трёхзначного числа на сотни, десятки и единицы
 • task13.c — Формат времени: перевод секунд в HH:MM:SS
 • task14.c — Исследование памяти: вывод адресов переменных и их разница
Инструкция по запуску
 1. Перейти в папку lab01:
cd lab01
 2. Скомпилировать выбранное задание с помощью GCC:
gcc task01.c -o task01
3. Запустить программу:
./task01
