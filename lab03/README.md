Лабораторная работа 3

Тема: Динамические массивы на языке С

Цель работы

Изучить работу с динамической памятью в языке C, включая функции:

malloc
calloc
realloc
free

Научиться создавать, изменять и освобождать динамические массивы.

Задание 1

Создание динамического массива с помощью malloc, ввод и вывод элементов.

Задание 2

Создание массива с помощью calloc и вывод значений (инициализация нулями).

Задание 3

Поиск максимального, минимального значения и суммы элементов массива.

Задание 4

Изменение размера массива с помощью realloc (увеличение и уменьшение).

Задание 5

Удаление элемента по индексу со сдвигом элементов.

Задание 6

Вставка элемента в заданную позицию.

Задание 7

Работа с динамическим массивом структур (Student).

Задание 8

Реализация динамического списка с меню (добавление, удаление, вывод).


Ответы на вопросы

Номер 1

malloc выделяет блок памяти заданного размера в байтах и возвращает указатель на него. Память не инициализируется.

Номер 2

malloc:

принимает 1 аргумент (размер)
не инициализирует память

calloc:

принимает 2 аргумента (количество элементов и размер)
заполняет память нулями

Номер 3

realloc изменяет размер ранее выделенного блока памяти. Может переместить данные в новое место и возвращает новый указатель.

Номмер 4

Память останется занятой до завершения программы, что может привести к нехватке памяти и замедлению работы.

Нмер 5

free освобождает ранее выделенную память, чтобы её можно было использовать повторно.

Номер 6

Программа может аварийно завершиться (ошибка), так как память освобождается дважды.

Номер 7

Нужно проверить указатель:
if (ptr == NULL) {
}

Номер 8

Статический массив:

фиксированный размер
создаётся автоматически
освобождается автоматически

Динамический массив:

размер задаётся во время выполнения
требует ручного управления памятью
можно изменять размер

Вывод

Динамические массивы позволяют эффективно использовать память и работать с данными переменного размера.
